# 变更日志

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 格式，
版本遵循 [语义化版本控制](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-06-24

### 新增
- 认证系统（JWT、登录、登出、用户管理、TOTP 二步验证）
- 系统监控仪表盘（CPU、内存、磁盘、网络实时监控）
- 服务管理（systemd 启停、日志查看、批量操作）
- Web 终端（xterm.js + PTY）
- 文件管理器（目录浏览、编辑、上传、下载）
- 部署同步（远程服务器管理、文件同步、命令执行）
- 运行环境管理（Node.js、Python、Go、Java 版本管理）
- 环境配置管理（.env 文件编辑）
- 网站管理（Nginx/Apache 配置、SSL 证书）
- 数据库管理（MySQL、PostgreSQL、MongoDB）
- 计划任务管理（Cron 表达式、脚本库）
- 防火墙管理（iptables 规则）
- SSH 管理（配置、会话、登录历史）
- 容器管理（Docker 容器、镜像、Compose）
- 腾讯云轻量服务器管理（实例、防火墙、快照）
- 操作审计日志（记录所有操作、导出 CSV）
- 面板设置（服务器、认证、监控、通知配置）
- 安全设置（IP 白名单、密码策略、会话管理）
- Webhook 通知（钉钉、飞书、企业微信）
- 监控告警（自定义规则、阈值告警）
- Docker 部署支持
- 安装脚本（一键部署到 Linux 服务器）

### 安全
- JWT 令牌认证与刷新机制
- 密码强度验证（大小写字母 + 数字）
- 登录失败锁定（可配置次数和时长）
- IP 白名单访问控制
- 令牌黑名单机制
- 单用户单会话限制
- CORS 安全配置
- 速率限制防护
